# 해외학회 & 가족여행 플래너 (conf-planner)

진단검사의학·혈액·병리·종양학 분야 해외 학회를 한곳에서 관리하는 정적 PWA. 서버·로그인 없이 브라우저 `localStorage`에만 저장됩니다.

## 기능
- **📋 학회**: 2026~2027 주요 학회 목록(진단검사·혈액·병리 + 임상종양·암연구). 분야·국가·연도·상태 필터, 검색, 날짜순/관심도순/국가순 정렬. 별점(관심도 0~5), D-day, 초록/얼리버드 마감, 공식 링크. `추정/미확정` 배지로 확정 안 된 일정 표시.
- **🧳 방문 준비**: `방문 예정`으로 표시한 학회마다 준비 체크리스트(초록·승인·등록·항공·숙소·비자·발표자료·보험), 항목 추가, 진행률, 예산·메모.
- **👨‍👩‍👧‍👦 가족여행**: 학회 일정에 붙이는 가족여행 계획. 동행 가족, 여행 기간(학회 날짜 자동 채움), 할 일, 예산, 메모.
- **🗓 타임라인**: 다가오는 마감·개막을 시간순으로.
- **백업/복원**: 내 데이터를 JSON으로 내보내기/불러오기(기기 이동용).

## 실행
```
python -m http.server 8742 --directory conf-planner
```
→ http://localhost:8742 . GitHub Pages 등 정적 호스팅에 그대로 배포 가능.

## 학회 데이터 수정
`data/conferences.json`의 `conferences` 배열을 편집. 앱 상단 `↻ 데이터`로 새로고침. 개인 데이터(관심도·체크리스트 등)는 학회 `id`로 연결되므로 id는 바꾸지 말 것.

> 등록·초록 마감은 조사 시점 기준. 투고/등록 전 공식 사이트에서 반드시 재확인.
